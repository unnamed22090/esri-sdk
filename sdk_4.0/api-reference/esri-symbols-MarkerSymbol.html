Class: MarkerSymbol

Properties | Methods

esri/symbols/MarkerSymbol

            require(["esri/symbols/MarkerSymbol"], function(MarkerSymbol) { /* code goes here */ });
Inheritance: MarkerSymbol Symbol Accessor
Subclasses: PictureMarkerSymbol , SimpleMarkerSymbol
Since: ArcGIS API for JavaScript 4.0

Marker symbols are used to draw Point graphics in a FeatureLayer or individual graphics in a 2D MapView. To create new marker symbols, use either SimpleMarkerSymbol or PictureMarkerSymbol.

Marker symbols may also be used to symbolize 2D polygon features. The image below depicts a FeatureLayer whose polygon features are styled with SimpleMarkerSymbols.

Marker symbols can be used in a 3D SceneView. However, it is recommended you use PointSymbol3D instead.

Property Details

angleNumber

The angle of the marker in degrees.

Default Value: 0

typeStringreadonly

The symbol type.

Possible values: simple-marker-symbol | picture-marker-symbol

xoffsetNumber autocast

The offset on the x-axis in points. This value may be autocast with a string expressing size in points or pixels (e.g. 12px).

Known Limitations

This property is not supported in 3D SceneViews.

Default Value: 0
Examples
// xoffset in points
symbol.xoffset = 4;
// xoffset in pixels
symbol.xoffset = "2px";
// xoffset in points
symbol.xoffset = "4pt";

yoffsetNumber autocast

The offset on the y-axis in points. This value may be autocast with a string expressing size in points or pixels (e.g. 12px).

Known Limitations

This property is not supported in 3D SceneViews.

Default Value: 0
Examples
// yoffset in points
symbol.yoffset = 4;
// yoffset in pixels
symbol.yoffset = "2px";
// yoffset in points
symbol.yoffset = "4pt";
